# FirewallX-Core 🔐

Smart Firewall and Intrusion Detection System (IDS) built from scratch.

## 🚀 Overview
This project captures live network traffic, applies firewall rules, and detects suspicious activity.

## 🎯 Features
- Packet capture using Scapy
- Firewall rule engine (allow/block traffic)
- Intrusion detection system (IDS)
- Logging and alert system

## 🛠️ Tech Stack
- Python
- Scapy
- Networking (TCP/IP)
